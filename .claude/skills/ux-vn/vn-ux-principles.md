# Skill: 베트남 사용자 UX 원칙

- 기준 질문: "베트남 중계인이 설명 없이 쓸 수 있는가?"
- 1화면 1작업, 버튼 3개 이하, 390px 모바일 기준
- 입력 위계: 사진 > 터치 토글 > 스테퍼 > 숫자패드 > (최후) 텍스트
- 라벨은 next-intl 키만 (vi 기본). 하드코딩 문자열 발견 즉시 수정
- 숫자 입력은 천단위 자동 콤마 + 베트남 통화 관습(50.000₫ 점 구분) 고려
- 공급자 화면 금지 항목: 판매가(KRW), 마진, 고객 연락처·국적 상세
- 업로드 UX: 카드 탭 → 카메라/갤러리 → 업로드 중 스피너 → 완료 체크. 실패 시 재시도 버튼
