# b14-zalo-chat — 디자인 노트 (ADR-0009 통합 리디자인)

## 화면 의도
운영자(ADMIN) 다크 Zalo 채팅. 기존 2-pane(인박스 320px + 대화창) · 9메뉴 사이드바 구조 유지. ADR-0009 D1~D9 반영.

## 변경 사항 (vs 기존 b14)
- **인박스**: 상대 타입 배지(공급자=teal / 고객=indigo / 미분류=slate) + 타입 필터 칩(전체/공급자/고객/미분류). 아바타는 `<img>`(있을 때) / 이니셜 원(폴백) 혼합 표시 (D8).
- **대화 헤더 리디자인 (D7·D8·D9 통합)**:
  - 아바타 이미지 + 이니셜 폴백 (D8)
  - 별명 + 연필 편집 아이콘 (D9), 아래 줄에 "Zalo 원본: …" 보존 표기
  - 상대 분류 배지(공급자) (D1)
  - 우측 번역언어 드롭다운: 끄기 / Tiếng Việt / English (D7), 현재 VI 선택 + 펼친 메뉴 디자인 참고
- **입력창 + 첨부 버튼 (D2)**: Composer 좌측 + 버튼 → 펼친 메뉴(공급자 변형: 사진 보내기/촬영/빌라 공유/정산서 공유). 제안서는 공급자 대화에 미노출.
- **공유 카드 버블 (D3·D5)**: 스레드에 image(사진), villa_share(빌라), settlement_share(정산) 카드 형태 추가. 색 코딩 — 빌라=blue 보더, 정산=amber 보더.
- **번역 OFF 상태 (D7.5)**: 디자인 참고 섹션에 미리보기 영역 없는 Composer(한국어끼리).
- **48시간 경과 상태**: 기존 유지.

## 누수 차단 (어느 카드에 어떤 필드)
- **빌라 공유 카드(공급자)**: 이름 · 사진 · 단지 · 침실/욕실 · 시설(수영장·조식) · **원가(2.500.000₫/박)**. 판매가·KRW·마진 없음. "원가 기준" 배지로 명시.
- **빌라 선택 모달(공급자)**: 원가만 표기. 고객 대화면 판매가만 표기(별도 변형은 b15에서 동일 컴포넌트 분기).
- **정산 카드(공급자 전용)**: 월 · 건수 · 상태 · 총 지급액(VND). 판매가·마진 없음.
- **제안 공유**: 고객 대화 전용(b14 공급자 변형 메뉴에서 제외). /p/[token] 공개 판매가 페이지 링크.
- **마진**: 어떤 카드·메뉴에도 없음.
- VND 표기: 운영자 화면이나 공급자에게 보내는 원가는 점 표기(2.500.000₫), 판매가/KRW 혼용 영역은 b15 고객 변형에서 쉼표.

## 상태색 정의
- 공급자 배지: teal `#2dd4bf` / 고객: indigo / 미분류: slate
- 빌라 공유 카드: blue 보더, 정산: amber 보더, 사진: blue 버블
- Zalo 연결됨: green dot

## 자가 채점 (DESIGN 평가 4기준)
- 디자인 품질 (상): 기존 #0F172A/#1E293B/blue 토큰·Noto Sans KR 폴백 그대로 — 신규 요소(카드·드롭다운·배지)가 동일 언어로 응집. 부품 나열 아님.
- 독창성 (중상): 공유 카드의 색 코딩(빌라=blue / 정산=amber)·"원가 기준" 배지로 누수 안전을 시각 언어화한 의도적 선택. AI 기본값 탈피.
- 완성도 (상): 타이포 위계(13/11/9px)·간격·tabular-nums·keep-all/nowrap 규칙 일관. 기존 b14와 픽셀 정합.
- 기능성 (상): +버튼→메뉴→모달→카드 흐름이 추측 없이 완결. 상대 타입별 메뉴 가시성으로 잘못된 공유 차단.
- **다듬기 vs 전환 판단**: 기존 b14 미학이 검증된 운영자 톤이므로 "현재 방향 다듬기" 선택. 신규 요소만 동일 토큰으로 추가, 미학 전환 불필요.

## 변환 인계 (FE)
- msgType 분기 렌더(image/villa_share/proposal_share/settlement_share) — 저장 text는 발송 본문 그대로(D3.5).
- 첨부 메뉴 가시성은 counterpartyType으로 클라 분기 + 서버 게이트 이중(D4.4).
- 번역 드롭다운 → PATCH SET_TRANSLATE_MODE, OFF면 미리보기 영역 미렌더.
- 별명 연필 → PATCH SET_NICKNAME. 아바타 img onError → 이니셜 폴백.
