# c3-booking-request — 가예약 입력 + 완료 확인

- Stitch screen: projects/14837850287160773673/screens/e8bc10e6517145d091d5a2e1ef037f3d
- 대상: 비로그인 한국 사용자 (ko), 모바일 우선, 라이트 신뢰감 (c1 톤), 브랜드 "Villa PMS"
- 디자인 의도: 한 화면에 두 상태를 참고용으로 표현 (구현 시 단계 전환):
  1. 입력 — 선택 빌라 요약 카드(사진/이름/날짜/총액 KRW 쉼표 표기) + 이름·연락처 폼 +
     "24~48시간 홀드" 안내 배너 + "가예약 신청하기" 버튼
  2. 완료 — 녹색 체크 + "가예약이 접수되었습니다" + 예약번호 칩(B-2611) +
     입금 안내 카드(은행/계좌/입금액/예금주) + 만료 카운트다운 배지("47:59 남음")
- 1차 출력의 "Villa Booking" 브랜드명 → "Villa PMS"로 다듬기 1회 수정
- 변환 메모(FE): 카운트다운은 홀드 만료 시각 기준 실시간, 폼은 react-hook-form + zod,
  가격은 가격 스냅샷(홀드 시점 고정) 값 사용
