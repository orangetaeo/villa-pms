1. **드로어**: 모든 `(admin)` 라우트에서 `<1024px` 햄버거 헤더 + 드로어가 동작(`sidebar.tsx` 경로 변경 시 자동 닫힘 유지). 회귀 없음.
2. **목록 카드 전환**: b5·b9·b12·b13 목록 화면이 `<768px`에서 가로 스크롤 없이 카드/1열로 렌더(코드 검증: 목록 표는 `ResponsiveTable` 사용 또는 반응형 그리드).
3. **잔여 가로 넘침 점검**: `(admin)` 내 **목록성 표** 중 `<768px`에서 의도치 않게 잘리는 것이 없음. 상세/특수 표(`bookings/[id]` 결제 기록, `checkout-form` 미니바, `settlements-view`, `season-manager`, `rate-editor`)는 `overflow-x-auto`로 가로 스크롤 허용 — 의도된 동작이므로 변경 대상 아님(발견 시 계약에 명시만).
4. **i18n 혼입 0**: 새로 추가한 텍스트는 ko/vi 양쪽 키 등재. 하드코딩 한국어/영어 라벨 없음.
5. `npm run typecheck` 통과 (에러 0).
6. `npm run lint` 통과 (에러 0; 기존 경고 수준 유지).
