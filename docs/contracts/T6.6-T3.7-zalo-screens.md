## 범위 (In)

### A. 운영자 `/messages` (b14 변환, T6.6)
1. `app/(admin)/messages/page.tsx` (RSC) — 2-pane. **좌 인박스**: ZaloConversation 목록(lastMessageAt desc, displayName·이니셜 아바타·마지막 메시지 truncate·시각/상대시간·unreadCount 뱃지·48h 경과 뱃지[lastInboundAt+48h<now]), 검색(공급자명). **우 대화창**: `?c=<conversationId>` 선택, 헤더(이름·"Zalo 연결됨" 배지[userId 매칭 시]·연결 빌라/단지), 메시지 스레드(날짜 구분자, INBOUND=vi 원문+ko 번역 병기[translatedText, "번역" 라벨+숨기기 토글], OUTBOUND USER/CHAT=ko 버블, SYSTEM=시스템 알림 미러 버블), 입력 푸터.
2. **48h 창 로직**: `lastInboundAt + 48h < now` → 입력창 비활성(amber 경고 배너 "48시간 경과 — 시스템 알림은 정상 발송"). 활성 시 ko 입력 + **베트남어 미리보기**(Gemini ko→vi, refresh 재번역) + 전송.
3. **발신 API** `POST /api/zalo/messages` (ADMIN 전용) — body `{ conversationId, text }`. 48h 가드(경과 시 409 WINDOW_CLOSED), ZaloMessage(OUTBOUND·CHAT·sentBy=세션 userId·status 초기 SENT or FAILED) 적재 + lib/zalo dispatch 시도(토큰 미설정/실패는 FAILED 기록·500 금지 — 영속은 성공), conversation.lastMessageAt 갱신, writeAuditLog. **마진·금액 미포함**.
4. **번역 미리보기 API** `POST /api/zalo/translate` (ADMIN) — `{ text, target: "vi" }` → Gemini 번역(미설정 503 폴백 — 입력창은 원문 전송 허용). lib/gemini 재사용 또는 translate 헬퍼.
5. **읽음 처리**: 대화 열람 시 `unreadCount=0` (PATCH `/api/zalo/conversations/[id]` 또는 page action). 멱등.
6. 사이드바 "메시지" 메뉴 + 미읽음 합계 뱃지.

### B. 공급자 Zalo 연결 (a0-zalo-connect 변환, T3.7 UI)
