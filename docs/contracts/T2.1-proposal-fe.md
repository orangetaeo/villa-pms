## 범위

1. **`/proposals/new`** (b2 변환, ADMIN 다크) — 3패널: 좌(날짜 범위·채널 선택), 중(판매 가능 빌라 카드 — 사진·이름·침실·수영장/조식 아이콘·1박 판매가·체크박스, 최대 3개 선택), 우(요약 — 고객명 입력, 유효기간 24h/48h 선택, 선택 빌라 스택, 총 판매가·마진 요약·최종 제안 금액, "링크 생성" 버튼)
   - 데이터: `GET /api/proposals/candidates?checkIn&checkOut&saleCurrency` (날짜·통화 변경 시 재조회), warnings(요율 미설정 빌라)는 안내 배너로 표시
   - **채널→통화 자동 전환(ADR-0003)**: 여행사/랜드사 선택 → 가격 전부 VND 쉼표 표기(9,800,000₫) + 캡션 "여행사·랜드사 채널은 VND로 결제됩니다", 직접 → KRW(1,280,000원). 하단 환율 참고 칩(`GET /api/settings`의 FX_VND_PER_KRW — 미설정 시 "환율 미설정")
   - 생성: `POST /api/proposals` → 201 시 생성 완료 화면/모달(공개 링크 `/p/{token}` 복사 버튼) → 목록 이동. 409 `items_unavailable`은 failures(빌라명·사유)를 그대로 안내
   - **유효기간은 24h/48h 선택만**(expiresInHours로 전송 — BE 계약: 제안별 holdHours 아님, "무제한" 금지)
   - 마진 요약: VND 채널 = 판매가−원가 정확값. KRW 채널 = fx 참고 환산(미설정 시 마진 행 숨김+안내) — **마진·원가는 ADMIN 화면이므로 표시 허용**
2. **`/proposals`** (b12 변환) — 상태 탭(전체/활성/사용됨/만료/회수 — effectiveStatus 기준), "+ 제안 만들기" 버튼, 테이블: 고객명, villa 칩, 날짜, 채널 배지, 생성일(YYYY.MM.DD), 만료 카운트다운(주황), 금액(채널 통화 혼재 — 우측 정렬·tabular-nums·nowrap), 상태 배지, 행 액션 [링크 복사]/[회수]
   - 데이터: `GET /api/proposals` (effectiveStatus 포함). <768px 카드 전환은 ResponsiveTable 컴포넌트 재사용(T6.7)
3. **`PATCH /api/proposals/[id]`** (신규 BE 소품) — `{action:"revoke"}`: 첫 줄 ADMIN 검사, ACTIVE(시각상 유효)만 REVOKED 전이(그 외 409), AuditLog. b12 회수 버튼용
4. 사이드바 "제안" 메뉴 활성 연결, i18n은 admin 화면 규약대로 ko 기본(nav/adminProposals 네임스페이스 키 추가)

## 범위 밖
- /p/[token] 공개 페이지(T2.2 병행 중), 제안별 홀드시간 스키마(TDA 백로그), 인원(guestCount) 저장 — **스키마에 필드 없음: b2의 인원 입력은 제외** (합의 편차, IDEAS 등재), 페이지네이션(GET take:100 — Phase 1 충분)

## 완료 기준 (테스트 가능)
1. ADMIN 로그인 → /proposals/new: 날짜+채널 선택 → 후보 카드 로드(사진·가격 표시) → 1~3개 선택 → 고객명·유효기간 입력 → 생성 → 토큰 링크 복사 가능, DB에 Proposal+items(+AuditLog)
2. 채널 전환 시 통화 표기 전환(여행사 ₫쉼표 ↔ 직접 원) + VND 결제 캡션 + 환율 칩 동작. ADMIN VND는 **쉼표** 표기 일관
3. 4개 초과 선택 불가(최대 3), 0개 선택 시 생성 버튼 disabled, 409 failures 안내 렌더
4. /proposals: 상태 탭 필터 동작(effectiveStatus 기준 — DB가 ACTIVE여도 만료 시각 경과면 "만료" 탭), 카운트다운 표시, 링크 복사, 회수 → REVOKED 반영
5. PATCH revoke: 비로그인 401/비ADMIN 403/ACTIVE 외 409/성공 시 AuditLog — 코드 검증
6. 공개 페이지가 아니므로 원가·마진 표시는 허용되나, **비ADMIN 세션이 이 라우트·API에 도달 불가** 확인(redirect/403)
7. 영어 잔존 0(b12 "Copy Link" 등 금지 — "링크 복사"), 날짜 YYYY.MM.DD, tsc·lint·build 통과, 충돌 구역 git diff 0
