   - PENDING·REJECTED만 제출 가능(상태기계 — REJECTED 재제출 시 rejectNote 상단 표시), PHOTOS_SUBMITTED·APPROVED는 읽기 전용(제출된 사진 그리드 보기)

## 완료 기준 (테스트 가능)
1. SUPPLIER 로그인 → /cleaning: 자기 빌라 태스크만, 상태 배지 4종·체크아웃일 표시. 타인 태스크 /cleaning/[id] 직접 접근 404
2. PENDING 태스크 → 전 공간 업로드(진행 카운터) 전 제출 버튼 disabled → 전부 업로드 후 제출 → PHOTOS_SUBMITTED 반영(목록 Đã gửi)

5. 가격·고객명·금액이 RSC 직렬화·DOM에 0건 (booking은 checkOut만)
