# 인스타그램 카피 가이드 — 톤앤매너 (v1, 2026-07-16)

> 담당: COPY 서브에이전트. 살아있는 문서 — 월 1회 트렌드 리서치로 갱신.
> 대상 독자: **한국인 여행객** (전부 한국어 카피). 베트남어 아님.
> 상위 원칙: 사업 핵심 원칙(재고 비공개·마진 비공개)과 충돌 금지. 이 문서의 문구는 오버레이 헤드라인·Gemini 캡션 프롬프트·CTA에 주입되어 매 포스트의 톤을 결정한다.

## 1. 브랜드 보이스

빌라고 푸꾸옥의 목소리는 **"먼저 다녀온 한국인 친구가 좋은 빌라를 소개해 주는 느낌"**이다.

| 축 | 지향 | 지양 |
|---|---|---|
| 온도 | 따뜻하고 다정함 — 가족·아이·부모님을 챙기는 시선 | 차갑고 사무적, 부동산 매물 나열 |
| 신뢰 | 담백한 사실 + "한국어 상담" 안심 포인트 | 과장·허풍·느낌표 남발 |
| 감성 | 여행의 한 장면을 그려주는 묘사("눈 뜨자마자 수영장") | 스펙만 나열("3BR/2BA 120㎡") |
| 격식 | **존댓말 기본** ("~하실 수 있어요", "~어때요?") | 반말·명령형("가라", "해라") |
| 길이 | 캡션 본문 3~6줄, 한 줄 한 호흡 | 한 문단에 정보 폭탄 |

### 이모지 규칙
- **캡션당 2~4개**로 절제. 문장 끝 장식용 1개 + 리스트 불릿용 정도.
- 권장 세트: 🌴(푸꾸옥·야자수) 💦🏊(수영장) 🏖(해변) 👨‍👩‍👧‍👦(가족) 🌅(뷰·일출) 💬📩(문의) ✔️(체크리스트).
- 금지: 🔥💰💯🤑 같은 **가격·과열·돈 암시 이모지** (마진 비공개 원칙·과장 광고 금칙과 충돌).
- 오버레이 헤드라인(이미지 위 텍스트)에는 이모지를 넣지 않는다 — 텍스트만 깔끔하게. 이모지는 하단 정보 바에만.

런타임은 Gemini가 이 뼈대 + 브랜드 보이스로 살을 붙이되, **구조·CTA·해시태그 슬롯은 고정**한다.

## 4. CTA 문구 사전

인스타 캡션엔 클릭 링크가 없다 → 전환 경로는 **프로필 링크** 또는 **DM** 둘뿐. 매 포스트 1개만 사용, 로테이션.

### 4-1. 프로필 링크 유도
```
- 📩 예약·견적 문의는 프로필 링크 클릭!
- 👆 프로필 링크에서 카톡으로 바로 상담하세요
- 🌴 자세한 빌라 정보는 프로필 링크 → 카카오톡에서
- 📩 날짜만 알려주시면 견적 드려요 — 프로필 링크 클릭
- 👆 마음에 드는 빌라, 프로필 링크에서 문의 남겨주세요
- 🗓️ 원하는 날짜 확인은 프로필 링크의 카톡 채널에서
```

### 4-2. DM 유도
```
- 💬 궁금한 점은 편하게 DM 주세요!
- 📩 "예약 문의" DM 남겨주시면 바로 안내드릴게요
- 💬 날짜와 인원만 DM 주시면 견적 보내드려요
- 📩 DM 주시면 카카오톡으로 빠르게 상담 도와드려요
- 💬 이 빌라 궁금하시면 DM으로 물어보세요
- 📩 가족 인원·일정 DM 주세요 — 맞춤 빌라 추천드려요
```

> 톤 주의: CTA도 존댓말·따뜻하게. "지금 바로!", "서두르세요" 같은 **압박·긴급성 조장 표현은 지양**(과장 광고 인접).

## 5. 금칙어·금지 표현 (기계 검사용)

승인 큐 진입 전 자동 필터. 아래 배열에 걸리면 **플래그 후 재작성**. 부분 문자열 포함 검사(대소문자·공백 무시) 기준.

### 5-1. 마진·원가 추정 (마진 비공개 원칙 — 최우선)
```json
["원가", "마진", "업체가", "업체 가격", "도매가", "떼오는", "떼어오는", "남는 게", "남기는", "수수료", "차익", "우리가 받는", "얼마에 가져와", "얼마에 사서"]
```

### 5-2. 확정 가격 박제 유도
```json
["1박 확정가", "정가 ", "무조건 이 가격", "가격 변동 없음", "이 가격 고정", "평생가", "박제", "특가 마감임박", "오늘까지만 이 가격"]
```
> 정책: 시작가는 "~부터"/"~" 표기만 허용(예: "1박 XX만원~"). 확정·고정·마감임박식 박제 문구는 금지.

### 5-3. 과장 광고
```json
["최저가 보장", "최저가", "업계 최저", "100%", "무조건", "절대", "완벽", "완벽한", "국내 유일", "유일무이", "최고의", "1등", "1위", "누구나 만족", "후회 없는", "강력 추천", "안 하면 손해", "여기가 끝판왕"]
```
> "최고", "완벽" 등은 정성 묘사로 흔히 쓰지만 인스타 과장 광고 리스크 → 대체 표현("넉넉한", "여유로운", "편안한") 사용.

### 5-4. 의료·효능 단정
```json
["치료", "완치", "효험", "효능", "질병", "통증 완화", "디톡스", "면역력", "건강 회복", "의학적", "부작용 없는"]
```
> 마사지·스파 서비스 카피에서 특히 주의. "편안하게 풀어주는", "여독을 씻는" 같은 **경험 묘사**까지만.

### 5-5. 초상권·재고 비공개 (콘텐츠 정책)
```json
["예약 가능 빌라 전체", "전체 리스트", "남은 빌라 목록", "공실 현황", "재고", "우리 보유 빌라 전부"]
```
> 포스팅은 개별 빌라 쇼케이스만. "현재 예약 가능한 빌라 전체"류 콘텐츠는 재고 비공개 원칙 위반.

## 5-6. 릴스·쇼츠 화면 자막 (영상 전용)

캐러셀 캡션과 **다른 규칙**이다. 화면에 1~3초 스치는 글자라 길면 못 읽는다.

**형식**
- **12자 이내**, 한 화면 한 문장. 조사 생략 가능("반세오 겉바속촉" O / "반세오는 겉이 바삭하고 속이 촉촉합니다" X)
- 마침표 금지. 느낌표는 편당 1회까지
- **컷마다 다른 문장** — 같은 문형 반복은 시청 이탈의 대표 원인이다
  (실측 2026-07-23: "○○, 이걸 먹으러 갑니다"가 3컷 연속 나가 테오 지적)

**컷 역할별 각도** (같은 각도라도 문장은 매번 새로 쓴다)
| 역할 | 각도 | 예시 |
|---|---|---|
| 외관·간판 | 어디인지 알려주기 | `여기가 메오키친`, `간판 보고 들어가면 됨` |
| 음식 | **그 음식의 한 가지 특징**만 | `겉바속촉 반미`, `반세오는 접어서`, `꼬치는 한 접시씩` |
| 내부 | 분위기 한 마디 | `안은 이런 느낌`, `자리는 넉넉` |
| 메뉴판 | 고르는 법 | `메뉴는 이 중에서`, `사진 보고 고르면 됨` |

**사실 규칙 (블로그와 동일)**
- 등록된 인상·사진 설명 밖의 사실을 만들지 않는다. 맛·재료·조리법을 추측하지 않는다
- 가격·영업시간·전화번호 금지 — **영상은 올린 뒤 못 고친다**

## 5-7. 블로그 본문 문체 (검색 유입용 긴 글)

인스타 캡션과 **다른 톤**이다. 스크롤하며 훑는 글이 아니라 검색해서 들어와 읽는 글이다.

- **인사말·자기소개로 시작하지 않는다** — "안녕하세요", "소개해 드릴게요"는 첫 화면을 낭비한다. 바로 본론
- 담백한 서술체. 광고 문구체 금지("입안 가득", "놓칠 수 없는", "안성맞춤", "망설임 없이")
- **판단을 쓴다**: 어떤 일행에게 맞는지, 언제 가면 좋은지, **안 맞는 경우도 한 번은 솔직히**
- 소제목은 지시문을 베끼지 말고 **내용 요약 명사구**로(번호·물음표 금지)
- 최상급은 **운영자가 직접 쓴 문장에 있을 때만** 인용. 카피라이터가 새로 만들지 않는다
- 사실 범위: 등록된 인상·메모·사진 설명까지. **위치 관계·재료·조리법·컨셉·역사**는 사실이므로 지어내면 폐기
- 가격·영업시간·휴무일·전화번호 금지(바뀌면 그 순간 틀린 글이 된다)

## 6. 교훈 축적

- (초판) 헤드라인은 스펙이 아니라 **장면**을 판다 — "3BR"가 아니라 "삼대가 모여도 넉넉한". 변수는 문장 안에 자연스럽게 녹인다.
- (초판) 티켓 공시가는 대중 노출 정보라 "공식가"까지는 언급 가능하나, **숫자 박제는 금지** — 나중에 가격이 바뀌면 발목 잡힌다(기획 §2-4).
