## 3. 첫 달 성공 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|---|---|---|
| 제안 → 가예약 전환율 | ≥ 30% | Proposal USED / 전체 발송 |
| 가예약 → 확정 전환율 | ≥ 50% | CONFIRMED / HOLD 생성 |
| 공급자 자력 등록 | 설명 없이 완료 | T4.4 + 파일럿 관찰 기록 |
| 청소 승인 소요 | 평균 < 24h | CleaningTask 타임스탬프 |
| 더블부킹 | **0건** | iCal 충돌 경보 로그 |
| Zalo 발송 성공률 | ≥ 95% | Notification SENT/전체 |

## 4. 오픈 후 개선 루프 (사업 운영 방식)

1. **수집**: 공급자 피드백은 Zalo(개별·단톡), 한국 여행사 피드백은 카톡 → 테오가 수집
2. **기록**: 모든 요청·아이디어는 IDEAS.md에 먼저 기록 — 즉시 구현 금지
3. **선별**: 격주 우선순위 회의(PM 주관)에서 IDEAS → TASKS 승격. 승격 기준 3문:
   - ① 예약이 더 늘어나는가? ② 공급자 이탈을 막는가? ③ 테오의 운영 시간을 줄이는가?
4. **축적**: 버그·교훈은 코드 수정과 함께 .claude/skills/ 해당 파일 "교훈 축적"에 패턴화
