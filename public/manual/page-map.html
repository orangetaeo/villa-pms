  • ✅ 최근 추가·배포 — 메시지 실시간 SSE(ADR-0024) · 비밀번호 자가 재설정(Zalo 코드)(ADR-0025) · 빌라 위치 지도 임베드(ADR-0026) · 공급자 직접판매(양방향)·공개 판매링크(ADR-0021, PR #48·#52) · 미니바 전환 자동 회수(RECOVER)(PR #59) · Railway cron 4종 등록·검증(알림·연체·로스터리마인더·주기청소).

🔴 꼭 필요한 기능 (필수)

✅ (완료) 파트너 청구서 + 수납 LEDGER 연결
청구서 탭·PDF(vi/ko/zh/ru)·Zalo 발송(PR #44·#57)에 이어 수납 → 복식부기 LEDGER 적재·정정(취소)까지 연결 완료(ADR-0027, PR #54·#58). 더 이상 잔여 없음.
1. Zalo 세션 끊김 경보 (S6)
알림 채널(zca-js QR 세션)이 끊기면 운영자에게 즉시 경보. 자동 재연결 유도.
왜 필수: 세션이 죽은 걸 모르면 예약·홀드만료·청소 알림이 통째로 누락되어 운영 마비.
2. Zalo 증빙 이미지 첨부 발송 (S5)
여권·동의서 서명 등 체크인 증빙 이미지를 Zalo로 첨부 전송.
왜 필수: 현재 텍스트 위주 — 증빙 사진 전달 경로가 닫혀 있음.
3. 러시아어(ru) 원어민 최종 감수
게스트 다통화/체크인 화면 ru 번역의 원어민 검수(현재 1차 기계감수).
왜 필수: ru 게스트 응대 중인데 어색한 번역은 신뢰·전환 저하. (자원 미확보 상태)

🟡 있으면 좋은 기능 (Phase 2 백로그)

  • 매출관리 페이지 /sales — 정산과 별개로 기간별 판매 실적 집계 (테오 2026-06-24 제안)
  • 통화별 마진 리포트 — KRW/VND 매출·마진 분리 대시보드
  • 미니바 소모 자동 차감 — 체크아웃 입력 → 재고 차감·보충 알림 (현재 수기, 전환 자동 회수 RECOVER만 완료 PR #59)
  • 환율 자동 갱신 cron — FX_VND_PER_KRW (현재 /settings 수동)
  • 예약 변경 기능 (현재 취소 후 재생성)
  • 빌라 위치 지도 — 전체 다중핀 보드 (개별 임베드 ADR-0026 완료, 좌표필드 도입 시 후속)
  • 품질점수 + 판매 후순위 정렬
  • 감사·알림 발송 로그 조회 화면 (현재 기록만)
  • SMS 폴백 · 여행사 월간 리포트 자동발송 · 캘린더 기간차단 UI · 관리자 하단탭
