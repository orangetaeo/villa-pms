🗺️ Villa Go 페이지 관계도

모든 화면(페이지)이 어떻게 이어지는지 — 실제 코드의 링크·라우팅·리다이렉트 기반 지도입니다.
🟣 관리자(운영) 🟢 공급자(빌라 주인) 🟠 손님(공개·비로그인) 🟪 원천공급자(VENDOR) ⬜ 인증·진입 ━━ 실선=화면 이동/링크   ┈┈ 점선=상태 전환/게이트

① 핵심 업무 파이프라인 존을 가로지르는 데이터 흐름

공급자가 올린 빌라 → 관리자가 판매 → 손님이 예약 → 체크인·아웃 → 정산. 색이 다른 박스는 서로 다른 역할의 화면입니다.
