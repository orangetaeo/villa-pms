# 릴스 BGM 라이선스 — 저작권 리스크 0 원칙

Villa PMS 인스타그램 릴스(lib/instagram/reels.ts)의 오디오 트랙은 **번들된 음원 파일을 사용하지 않는다.**
저작권 분쟁 리스크를 원천 제거하기 위해, 오디오는 두 방식 중 하나로 **런타임에 ffmpeg가 직접 생성**한다.

## 1. 기본값 — 무음 트랙 (`audio: "silent"`)

- ffmpeg `anullsrc`(무음 소스)로 AAC 오디오 스트림을 생성한다.
- 컨테이너에는 항상 H.264(비디오) + AAC(오디오)가 존재해 인스타그램 릴스 규격을 충족하되, 실제 소리는 없다.
- 운영자가 인스타그램 앱에서 **트렌드 음원**을 직접 얹기에 가장 유리하다(도달률에 유리한 실무 관행).
- 저작권: 무음이므로 해당 없음.

## 2. 옵션 — 합성 앰비언트 패드 (`audio: "ambient"`)

- 번들 음원이 아니라 ffmpeg `aevalsrc` 표현식으로 **직접 합성한** 순음(sine) 화음이다.
  - 구성: G3(196Hz) + C4(261.63Hz) + E4(329.63Hz) 저음량 C장조풍 패드 + 인/아웃 페이드.
- 어떤 저작물의 샘플·멜로디·녹음도 포함하지 않는다. 순수 수학적 파형 합성 결과물이다.
- 저작권: 저작물성이 없는 단순 순음 합성 → **퍼블릭 도메인에 준함(CC0 취급)**. 제3자 권리 없음.

## 결론

릴스 파이프라인은 외부 음원 파일을 배포하지 않으므로 이 디렉터리에 오디오 바이너리가 존재하지 않는다.
향후 확실한 무료(CC0/직접 제작) 트랙을 번들하려면, 그 파일과 함께 출처·라이선스 전문을 이 문서에 추가하고
`lib/instagram/reels.ts`의 오디오 입력 경로를 확장할 것.
